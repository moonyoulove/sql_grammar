平均值。如果沒有選取任何資料列，結果為 NULL。聚合僅允許在 SELECT 敘述中使用。傳回值的資料類型與參數相同。

範例：

AVG(X)

PERCENTILE_CONT

DESC
)

資料行中值的第 n 個百分位數。百分位數值可介於 0 到 1 之間（包含 0 和 1）。聚合僅允許在 SELECT 敘述中使用。傳回值的資料類型為 decimal。

範例：

PERCENTILE_CONT( 0.9 ) WITHIN GROUP (ORDER BY X ASC)

PERCENTILE_DISC

ASC
DESC
)

PERCENTILE_DISC 是一個假設為離散分佈模型的反分佈函數。它需要一個百分位數值和排序規範，並從集合中傳回一個元素。計數中會忽略 Null 值。

範例：

PERCENTILE_DISC( 0.9 ) WITHIN GROUP (ORDER BY X DESC)

範例：

PERCENT_RANK( 100 ) WITHIN GROUP (ORDER BY X ASC)

TO_NUMBER

TO_NUMBER(  |  [, ] )
TO_NUMBER ( stringTerm
timeStampTerm
 
, formatString
)

將字串或 date/time/timeStamp 格式化為數字，並可選擇性接受一個格式化字串。關於格式的詳細資訊，請參閱 java.text.DecimalFormat。對於 date、time 和 timeStamp 用語，結果為自 epoch 起計算的毫秒時間。此方法會傳回一個小數 (decimal number)。

範例：

TO_NUMBER('$123.33', '\u00A4###.##')

COALESCE

COALESCE( ,  )
COALESCE ( firstTerm , secondTerm )

如果第一個參數非 null 則傳回該值，否則傳回第二個參數的值。可用於確保 UPSERT SELECT 命令中的資料行會被評估為非 null 值。

範例：

COALESCE(last_update_date, CURRENT_DATE())

SUBSTR

SUBSTR( ,  [,  ] )
SUBSTR ( stringTerm , startInt
 
, lengthInt
)

傳回從 1 為基底 (one-based) 位置開始的子字串。如果使用 0，則位置是以 0 為基底 (zero-based)。如果起始索引為負數，則起始索引是相對於字串的結尾。長度是選填的，如果未提供，將傳回字串的剩餘部分。

範例：

SUBSTR('[Hello]', 2, 5)
SUBSTR('Hello World', -5)

REGEXP_SUBSTR

REGEXP_SUBSTR( ,  [,  ] )
REGEXP_SUBSTR ( stringTerm , patternString
 
, startInt
)

套用正則表達式，自以 1 為基底的偏移位置開始傳回子字串。就像 SUBSTR 一樣，如果起始索引為負數，則它是相對於字串的結尾。如果不指定，起始索引預設為 1。

範例：

REGEXP_SUBSTR('na1-appsrv35-sj35', '[^-]+') evaluates to 'na1'

REGEXP_REPLACE

REGEXP_REPLACE( ,  [,  ] )
REGEXP_REPLACE ( stringTerm , patternString
 
, replacementString
)

套用正則表達式並將符合的部分取代為替換字串，然後傳回一個字串。如果未指定替換字串，則預設為空字串。

範例：

REGEXP_REPLACE('abc123ABC', '[0-9]+', '#') evaluates to 'abc#ABC'

TO_CHAR

TO_CHAR(  |  [, ] )
TO_CHAR ( timestampTerm
numberTerm
 
, formatString
)

將 date、time、timestamp 或數字格式化為字串。預設的日期格式是 yyyy-MM-dd HH:mm:ss 並且預設的數字格式是 #,##0.###。詳細資訊請參閱對於日期/時間值的 java.text.SimpleDateFormat 以及對於數字的 java.text.DecimalFormat。此方法會傳回一個字串。

範例：

TO_CHAR(myDate, '2001-02-03 04:05:06')
TO_CHAR(myDecimal, '#,##0.###')

ROUND

ROUND(, {'DAY' | 'HOUR' | 'MINUTE' | 'SECOND' | 'MILLISECOND'} [, ])
ROUND ( timestampTerm ,
' DAY '
' HOUR '
' MINUTE '
' SECOND '
' MILLISECOND '
 
, multiplierNumber
)

將時間戳記四捨五入到最近的指定時間單位。乘數用於四捨五入到時間單位的倍數（例如：10 分鐘），如果未指定則預設為 1。此方法會傳回一個 date。

範例：

ROUND(date, 'MINUTE', 30)
ROUND(time, 'HOUR')

TRUNCATE

TRUNC(, {'DAY' | 'HOUR' | 'MINUTE' | 'SECOND' | 'MILLISECOND'} [, ])
TRUNC ( timestampTerm ,
' DAY '
' HOUR '
' MINUTE '
' SECOND '
' MILLISECOND '
 
, multiplierInt
)

將時間戳記截斷到更接近 0 的下一個時間單位。乘數用於截斷至時間單位的倍數（例如：10 分鐘），如果未指定則預設為 1。此方法會傳回一個 date。

範例：

TRUNCATE(timestamp, 'SECOND', 30)
TRUNCATE(date, 'DAY', 7)

TO_DATE

TO_DATE(  [, ] )
TO_DATE ( stringTerm
 
, formatString
)

解析字串並傳回 date。最重要的格式字元為：y 年，M 月，d 日，H 時，m 分，s 秒。預設格式字串為 yyyy-MM-dd HH:mm:ss。格式詳細資訊請參見 java.text.SimpleDateFormat。

範例：

TO_DATE('Sat, 3 Feb 2001 03:05:06 GMT', 'EEE, d MMM yyyy HH:mm:ss z')

CURRENT_DATE()
