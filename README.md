# Phoenix 語言參考文件 (繁體中文版)

這份文件包含了 [Apache Phoenix](https://github.com/forcedotcom/phoenix) 官方 SQL 參考文件的繁體中文翻譯，目的是為了方便中文使用者快速查詢 Phoenix 的語法、函數與資料類型。

## 說明

- **翻譯工具**：本專案所有的 HTML 頁面文件皆是由 **[Gemini](https://gemini.google.com/)** 協助進行繁體中文翻譯。
- **原始專案**：Apache Phoenix 是一個建構在 HBase 之上的 SQL 層。原始專案網址為 [https://github.com/forcedotcom/phoenix](https://github.com/forcedotcom/phoenix)。

## 文件目錄

- [SQL 語法 (SQL Grammar)](index.html) - 提供頂層命令及 SQL 語法的詳細說明。
- [函數 (Functions)](functions.html) - 整理了 Phoenix 內建的各種類型函數及其用法。
- [資料類型 (Data Types)](datatypes.html) - 列出並說明 Phoenix 支援的資料類型。
